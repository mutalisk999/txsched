Installing txsched
==================


On Linux:

    pip install Twisted==13.1.0


On windows:

    pip install Twisted==13.1.0
    pip install pywin32==220


If you want to use the SNMP testcase also, you still should to install:

    pip install TwistedSNMP-working==0.3.13
    pip install pysnmp-se==3.5.2
